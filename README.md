### Prerequisites

#### Cloning the repository

```
git clone https://github.com/harshOpensource/spotify-clone.git
```

### Install packages

```
npm i

```

#### Setup .env file

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

```

### Start the App
```
npx run dev
```
